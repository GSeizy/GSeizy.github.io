Troubleshooting Help

Sometimes things don't go as planned with your build. That's okay though as most issues have simple solutions. Here are some common problems and ways to fix them.

Common Problems

PC won't turn on

  • Check that the power supply switch on the back is set to "I" (on) instead of "O" (off)
  • Make sure the power cable is plugged into the wall and the PSU and is not loose at all at either port
  • Verify the 24-pin motherboard connector is fully seated and has no bent pins
  • Check that the power button is connected correctly and the cables to it are snug and intact

PC turns on but no display

  • Make sure your monitor is connected to the GPU input slots, not the motherboard slots
  • Check that the GPU is fully seated in the PCIe slot
  • Verify the GPU power cables are connected snugly
  • Ensure RAM sticks are clicked in and seated properly
  • Check that latest GPU drivers are installed

PC keeps restarting or crashes

  • Check CPU cooler is mounted properly and has thermal paste
  • Make sure all power connections are secure
  • Test with one RAM stick at a time to rule out faulty memory
  • Check for any signs of overheating (broken fans, cooling, etc...)

BIOS doesn't detect storage drive

  • Check SATA data and power cables are connected
  • Try a different SATA port on the motherboard, you may have a bad one
  • For M.2 drives, make sure it's in the correct slot and seated properly

USB ports not working

  • Check that the USB headers are connected to the motherboard
  • Try the ports on the back of the motherboard (if they work, it's likely a connection issue)
  • Install motherboard drivers after Windows is set up

Bought used parts? Extra tips

  • Always test used parts as soon as possible! Especially depending on your return window
  • If a used GPU crashes or shows visual artifacts, it is a likely sign that it is beginning to fail
  • Used PSUs can be risky, if your system is unstable, the PSU is a good place to check
  • Check for bent pins on used CPU's and motherboards before installing
  • Reset the BIOS to default settings on used motherboards
  • Always clean out any dust from used components BEFORE installing them
  • If buying used RAM, run a memory test in the BIOS (like MemTest86 via a USB drive from PassMark)
